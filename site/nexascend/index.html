    🚀 NexAscend — Sovereign Invocation Framework

    NexAscend is an ethically-driven, modular invocation system that transforms low-spec devices into powerful cloud-controlling terminals. Through full GUI integration, CLI orchestration, decentralized agent triggers, and identity-aware execution, it empowers sovereign-grade deployment from anywhere, by anyone, responsibly.

    🌟 Vision

    "Technology should be sovereign, ethical, and accessible."
    – MJ Ahmad, Founder of NexAscend

    NexAscend is not just software — it’s a principle. It ensures that every line of code serves purpose, transparency, and accountability in distributed environments.

    📐 Architecture Overview

    nexascend-core/
├── controller/         # CLI tool (`nexascend-controller.py`)
├── launcher/           # GUI launcher (Electron wrapper)
├── bootstrap/          # VM provisioning & automation scripts
├── docs/               # MkDocs documentation
├── .github/workflows/  # CI deployment for docs
├── .devcontainer/      # VS Code Dev Container config
├── LICENSE             # Ethical Invocation License
└── README.md           # Project overview

    ⚙️ Core Features

    • 🌐 Remote invocation via Hetzner or Azure VMs
    • 🧩 Module execution from CLI or GUI (Python/Node agents)
    • 🖥️ GUI-ready Ubuntu environment (XFCE + xrdp)
    • 📚 MkDocs-powered documentation auto-deployed to GitHub Pages
    • 🔐 Sovereign identity handshake using LumID (configurable)
    • 🪢 Integration with decentralized storages like Storacha & Web3.Storage

    🚀 Getting Started

    1️⃣ Clone This Repo

    git clone https://github.com/mj-nexara/core.git
cd core/

    2️⃣ Provision Cloud VM (Ubuntu 22.04 LTS)

    Run the automated bootstrap script:

    bash bootstrap/bootstrap.sh

    3️⃣ Launch CLI Controller

    python3 controller/nexascend-controller.py

    4️⃣ Access GUI Remotely

    • Use RDP Client to connect to your VM IP at port 3389
    • Desktop: XFCE4
    • Credentials: Configured via bootstrap script
